JavaScript基础测试

JavaScript是一门很独特的程序语言，非常的灵活，但经常会有一些让人难以理解的“坑”存在。国外有很多针对这些难以理解的点编辑的测验题，帮助新手们更好地理解这些“坑”背后的原理。我从中精选了12个自认为比较重要测验点，供各位参考学习。
同时，针对这些题目也写了一篇解答文章，JavaScript基础测试题解答，大家在测验完成后，可以对着解答详细了解这些点对应的原理。请确认每个题目都有答案被选中，未选题目按照答错处理。

  • 1. 如下代码返回的结果为？

    function foo() {
    alert(x);
    var x = 10;
}
foo();

  • 2. 如下代码返回的结果为？

    function foo() {
    alert(x);
    x = 10;
}
foo();

  • 3. 如下代码返回的结果为？

    function bar(){
	foo();

	function foo(){
		alert(99);
	};
}
bar();

  • 4. 如下代码返回的结果为？

    function foo(){
	for (var i = 0; i < 5; i++){
		alert(i);
	}
	alert(i);
}
foo();

  • 5. 以下结果返回true还是false？

    function Foo(){
	var bestBand = 'Jimmy Eat World';
}
Foo.prototype.tellMe = function(){
    alert(this.bestBand);
};

var f1 = new Foo();
var f2 = new Foo();
alert(f1.tellMe === f2.tellMe);
