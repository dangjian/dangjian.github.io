1. 对象

    // 1. 对象（object）是一种数据结构，由若干个“键值对”（key-value）构成。大括号就代表一个对象，被赋值给变量o。这个对象内部包含一个键值对（又称为“成员”），p是“键名”（成员的名称），字符串“Hello World”是“键值”（成员的值）
    var o = {
        p: "Hello World"
    };
    // 键名加不加引号都可以，上面的代码也可以写成下面这样：
    var o = {
        "p": "Hello World"
    };
    // “键名”又称为“属性”（property），它的“键值”可以是任何数据类型。如果一个属性的值为函数，通常把这个属性称为“方法”。
    var obj = {
        foo: 123,
        bar: function () { ... }
    }

    // 2. 生成方法: 对象的生成方法，除了像上面那样直接使用{}，还可以用new Object()命令。
    var o = {};
    // or
    var o = new Object();

    // 3. 读取属性: 读取一个属性，有两种方法，一种是使用点运算符，还有一种是使用方括号运算符。
    var o = {
        p: "Hello World"
    };
    // 点运算符
    o.p
    // "Hello World"
    // 方括号运算符: 如果使用方括号，键名必须放在引号里面，否则会被当作变量处理。
    o["p"]
    // "Hello World"
    o[p]
    // undefined

    // 4. 赋值属性：点运算符和方括号运算符，不仅可以用来读取值，还可以用来赋值。
    o.p = "abc";
    o["p"] = "abc";

    // 5. 属性的增加与删除
    // JavaScript允许属性的“后绑定”，也就是说，你可以在任意时刻新增属性，没必要在定义对象的时候，就定义好属性。
    var o = { p:1 };
    // 等价于
    var o = {};
    o.p = 1;
    // delete命令可以删除属性。
    var o = { p:1 };
    delete o.p
    // true
    o.p
    // undefined http://jsfiddle.net/dangjian/7suTd/

    // 6. 对象的引用: 如果不同的变量名指向同一个对象，那么它们都是这个对象的引用，也就是说指向同一个内存地址。修改其中一个变量，会影响到其他所有变量。
    var v1 = {};
    var v2 = v1;
    v1.a = 1;
    v2.a
    // 1 http://jsfiddle.net/dangjian/hrA7p/

    // 7. in运算符: in运算符用于确定某个属性是否包含在对象中。如果某个属性值是undefined，in运算符也返回true。
    var o = { p: 1 };
    'p' in o
    // true
    var o = { p: undefined };
    'p' in o
    // true

    // "Hell"

8. JSON(JavaScript Object Notation)

  • 格式1： “名称/值”对的集合（A collection of name/value pairs）。在JavaScript中，它被理解为对象（object）

    对象是一个无序的“‘名称/值’对”集合。一个对象以“{”（左括号）开始，“}”（右括号）结束。每个“名称”后跟一个“:”（冒号）；“‘名称/值’ 对”之间使用“,”（逗号）分隔。

  • 格式2：值的有序列表（An ordered list of values）。在JavaScript中，它被理解为数组（array）。

    数组是值（value）的有序集合。一个数组以“[”（左中括号）开始，“]”（右中括号）结束。值之间使用“,”（逗号）分隔。

  • 值（value）可以是双引号括起来的字符串（string）、数值(number)、true、false、 null、对象（object）或者数组（array）。这些结构可以嵌套。 
    // 1. 格式正确的JSON定义

    // 1. Date对象的生成

    // 对Date对象使用new命令，则生成一个代表时间的实例对象。如果不提供参数，生成的就是代表当前时间的对象。
    var today = new Date();
    today
    // "Sat Mar 09 2013 08:46:54 GMT+0800 (CST)"
    // 如果提供参数，则生成一个代表指定时间的对象。
    // 日期字符串: 所有可以被Date.parse()方法解析的日期字符串，都可以当作Date对象的参数。
    var date1 = new Date ( "January 6, 2013" );
    date1
    // "Sun Jan 06 2013 00:00:00 GMT+0800 (CST)"
    new Date ( "2013-02-15" )
    // Fri Feb 15 2013 08:00:00 GMT+0800 (CST)
    new Date ( "2013-FEB-15" )
    // Fri Feb 15 2013 08:00:00 GMT+0800 (CST)
    // 独立参数：除了日期字符串，Date还接受年、月、日等变量独立作为参数。月份是从0开始计算的，所以2就表示三月。
    new Date(year, month, day [, hour, minute, second, millisecond]);
    new Date(2013,2,1)
    // Fri Mar 01 2013 00:00:00 GMT+0800 (CST)
    // 毫秒时间戳: Date对象还接受从1970年1月1日00:00:00 UTC开始计算的毫秒数作为参数。
    new Date(1378218728000)

    Math.floor(3.2) // 3

    // 4. ceil方法：该方法返回大于参数值的最小整数。
    Math.ceil(3.2) // 4
    Math.ceil(-3.2) // -3

    // 5. Max方法：Max方法返回最大的那个参数。
    Math.max(2, -1, 5)
    // 5

    // 6. pow方法：该方法返回以第一个参数为底数、第二个参数为幂的指数值。
    Math.pow(2, 2) // 4
    Math.pow(2, 3) // 8

    // 7. sqrt方法：该方法返回参数值的平方根。如果参数是一个负值，则返回NaN。
    Math.sqrt(4) // 2
    Math.sqrt(-4) // NaN

    // 8. random方法：该方法返回0到1之间的一个伪随机数，可能等于0，但是一定小于1。
    Math.random() // 0.7151307314634323
    // 返回给定范围内的随机数
    function getRandomArbitrary(min, max) {
        return Math.random() * (max - min) + min;
    }
    // 返回给定范围内的随机整数
    function getRandomInt(min, max) {
        return Math.floor(Math.random() * (max - min + 1)) + min;
    }
